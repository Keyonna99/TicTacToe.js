# Tic Tac Toe Game

## Description
A web-based Tic Tac Toe game built with JavaScript, HTML, CSS.
Players can play against each other on the same computer.
This project demostrates interactive web development, game logic, and basic user interface design. 

## Features
- Two-player gameplay on the same device
- Interactive 3x3 game board
- Highlight winning line
- Reset button to start a new game

## Technologies Used
- HTML
- CSS
- JavaScript
